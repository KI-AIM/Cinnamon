Date Format - Information

Explanation

The date format specifies how a date value is written in the dataset and must be provided for date attributes. This prevents errors when parsing dates like mixing up day and month. All values in the dataset must follow this format, otherwise they will be considered invalid.
The date format is specified by pattern letters that consists of literal characters. Below you can find a selection of the most important patterns and examples for commonly used formats. The capitalization of the letters is important, because the lowercase and uppercase variants of a letter are often used for different patterns.
