String Pattern - Information

Explanation

The string pattern defines the expected structure or format of a text value. Providing this information prevents typos and ensures uniform formats for attributes like patient IDs, diagnosis codes, or postal codes. A value is considered valid if the entire string matches the pattern. Below you can find some examples for common regular expressions that should give you a good idea of what you can do with regular expressions. For more information about regular expressions, please see this page: https://www.regular-expressions.info/

Examples
Pattern Explanation Elements
.* Matches any textual value of any length
. Matches any character
* The preceding expression may occur zero or more times
\d{5} Matches a number with 5 digits
\d Matches one digit
{n} The preceding expression may occur exactly n times
(A|B|AB|O) Matches the blood types
A Matches the character "A"
( | ) Represents an "or" and matches one of the values seperated by "|" inside the brackets.
